Habit Tracker

A simple and responsive Habit Tracker web application that helps users build and maintain daily habits. Users can add, update, track, and manage their habits efficiently through an intuitive interface.

🚀 Features
Add new habits
Mark habits as completed
Update and delete habits
Track daily progress
Responsive and user-friendly design
Store and manage habit data using MongoDB
🛠️ Tech Stack
Frontend
HTML
CSS
JavaScript
Backend
Node.js
Express.js
MongoDB
📌 About the Project

This project was built to practice full-stack web development and demonstrate how to create a habit management application using modern web technologies. The frontend provides a clean user interface, while MongoDB is used to store and manage user habit data efficiently.
